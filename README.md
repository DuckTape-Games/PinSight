PinSight


PinSight is a Python-based desktop application with plans for web and mobile versions for tracking and analyzing bowling performance at a granular level. It allows bowlers to log sessions, games, frames, and individual rolls, providing deeper insight into scoring trends, shot selection, and equipment usage over time.


Unlike basic score trackers, PinSight supports roll-level data collection, enabling analysis of:

●Ball usage (including mid-frame switches)

●Hook vs. straight shot tendencies

●Frame-by-frame performance

●Long-term improvement trends


The application is designed to support both casual tracking and advanced analytics, allowing users to log as much or as little detail as they want while still gaining meaningful insights into their game.


Key Features:

●Track bowling sessions, games, and detailed frame/roll data

●Record equipment usage per roll (ball selection)

●Analyze performance trends over time

●Compare results across different pin systems (e.g., string vs free-fall)

●View historical data and session summaries

●Visualize progress with performance graphs

●Purpose


PinSight was built to go beyond simple scorekeeping by providing a structured way to analyze how different factors—such as equipment, shot type, and conditions—impact performance. It is intended for bowlers who want to better understand their game and make data-driven improvements.


Tech Stack:

●Language: Python

●Database: SQLite (sqlite3)

●UI Framework: Flet

●Data Analysis: Pandas

●Visualization: Matplotlib

*Additional standard libraries are used for date handling, data validation, and application logic.
